COŢOBĂNÍ vb. refl. (Mold., Munt., Olt.) „a se cocoța; a se îndârji, a se strădui; a înfrunta, a se rățoi – se hisser; s’obstiner, s’efforcer; faire le fier, monter sur ses grands chevaux” 1893 (Şez. II 224), var. (Olt., Munt.) coțopăni, (Vâlcea) coțopeni, cuțupeni „a se strădui, a face un efort mare – s’efforcer”.

Probabil ucr. kocubytycja „a se îndărătnici, a se îndârji” º (SDLR: v.sl. kobacati, ucr. dial. kacabatysja „a face tumbe”, scr. kobacati se „a tropăi”), cu sensurile „a se cocoța” și „a se rățoi” infl. de a se coțofăni/coțofeni º // ← coțob TDRG; cf. coțobâră, coțobăi DA; ← coțăi CDER 2483.
