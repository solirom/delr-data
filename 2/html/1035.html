CLASIBÓCRU s.m. (Mar., hapax, Viciu, Gl. 90) „cârlig de fier cu care se trag plutele, țapină – crochet de fer pour tirer les radeaux, gaffe” 1906.

Et. nec.; a doua parte a cuvântului este probabil rom. bocor „plută, bușteni legați împreună” (< magh. bokor „tufă, mănunchi, grămadă” Tamás, EW, 131-132) // Probabil din radicalul verbului slav klěsnǫti „a prinde, a lega” + ucr. bokory „plute” N. Drăganu, DR 3, 1922-1923, 704 (dar ucr. bokor provine din magh. sau rom. potrivit ESUM, iar verbul klěsnǫti apare în ucr. în forma klisnyty).
